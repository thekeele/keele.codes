Minimal Mean Seed

Minimal Mean Seed is a lightweight version of the, mean seed. The MEAN seed is a fantastic all JavaScript web stack similar to LAMP or MAMP. There are some great full-fledged seeds, like ultimate seed, but this is for developers looking for a simple start. This seed connects to a MongoDB database, creates an ExpressJS app, starts a NodeJS RESTful API, and serves a single page AngularJS app. Twitter Bootstrap 3 and Material Design are used to create a slick, modern interface.

Lux Ex Machina

Lux Ex Machina, the light machine, is an attempt to aid photographers by suggesting aperture and shutter speed based on a given amount of lumens. This app was designed as an in-browser phone app that pairs with an embedded ARM system. The embedded system would measure lumens and POST the results to an API that the web app could query. There's a lot of moving pieces here so check out the Prezi for more detail. If you're feeling curious, click the preview on the right and have fun with the sliders.

OSX Talkbox

Did you know your MacBook could talk to you? No, I'm not referencing the infamous Siri. The Darwin kernel has a system command, say, that can be leveraged to verbally express your feelings. Talkbox is a simple Ruby script that allows the user to say anything with a specific voice and volume. This app was created out of pure curiosity and can help with accessibility or to just have some old fashioned fun.

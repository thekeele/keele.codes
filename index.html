Minimal Mean Seed

Minimal Mean Seed is a lightweight version of the, mean seed. The MEAN seed is a fantastic all javascript web stack similar to LAMP or MAMP. There are some great full-fledged seeds, like ultimate seed, but this is for developers looking for a simple start. This seed connects to a mongodb database, creates an express app, starts a nodejs RESTful API, and serves a single page AngularJS app. Twitter Bootstrap 3 and Material Design are used to create a slick, modern interface.

Lux Ex Machina

The light machine is an attempt to aid photographers by suggesting aperture and shutter speed based on a given amount of lumens. This app was designed as a in-browser phone app that would read lumens from a sensor running on an STM-32 board. Initially, when the app boots up it will suggest an aperture and shutter speed. Moving a slider will change the aperture (or shutter speed) and adjust the other setting so that a light balanced photo could be taken with the settings. A warning system is in place so that the user will be alerted when settings are computed to be out of balance. The goal is to provide a functionality similar to the Auto mode on DSLRs for manual cameras.

OSX Talkbox

Did you know your MacBook can talk to you? No, I'm not referencing the famous Siri. The Darwin kernel has a system command, say, that can be leveraged to verbally express your feelings. Talkbox is a simple Ruby script that allows the user to say anything with a specific voice and volume. This app was created out of pure curiosity and can help with accessibility or to just have some old fashioned fun.

Streaming Sentiment

The idea behind the Streaming Sentiment app is to stream tweets from Twitter, with a given hashtag, and then analyze the sentiment of each tweet. This app was inspired by a paper that used tweets from Twitter to predict swings in the stock market. This is my latest project and still under development so be on the lookout for updates on my blog.
